# Nix Extra Configuration

This directory is for extra Nix configuration files intended for use with `--orbstack-nix-config` that is ignored by
Git, so non-maintainers of this repository can conveniently keep their extra Nix configuration in the repository
directory.

If you want to track your extra Nix configuration in Git, make another repository and reference the path of
your Nix configuration file with `--orbstack-nix-config`.
